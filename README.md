# LLM APP

Description: 
 - Create a Flask App that will allow users to chat with a Language Model.
 - User Login (Admin will register the user)
 - LLM Model
 - Chat should be persistent

APIs
-----

1. User Login -> /login
2. Start Chat -> /chat
3. Previous Chat -> /previous_chat


Models
------

1. User
2. Chats
3. LLM Model
4. Message
